Documentation for movie api

Description of Endpoint Return Endpoint URL Method Type Format of Request Data Format of Response Data
Return a list of ALL movies /movies GET None A JSON object holding data about all movies
Return data about a single movie by title /movies/:title GET None A JSON object holding data about the requested movie ex:

{ title: 'Avengers: Endgame',
description: "After the devastating events of Avengers: Infinity War, the universe is in ruins. With the help of remaining allies, the Avengers assemble once more in order to reverse Thanos' actions and restore balance to the universe.",
genre: 'Action',
director: {
name: 'Anthony Russo, Joe Russo',
bio: 'Collectively known as the Russo brothers are American directors, producers, and screenwriters.',
birth_year: '1970, 1971' }, },
Return data about a single movie's genre by title /movies/:title/genre GET None A JSON object holding data about the genre of a specific movie ex: "Sci-Fi"
Return data about a single movie's director by title /movies/:title/director GET None A JSON object holding data about the director of a specific movie:
{
"name": "Christoper Nolan",
"bio": "Christopher Nolan is an Academy Award-winning movie director and screenwriter who's helmed several hit films, including Inception, The Dark Knight, Interstellar, and Oppenheimer.",
"birth_year": "1970"
}
Return the documentation page /documentation GET None A html page holding the documentation page of this app
Create new user /users POST A JSON object holding data from the user to be added:
{
"name": "Chris",
"favMovies": "Inception"
} 		A JSON object holding data from the user that has been added plus uuid: {
"name": "Chris",
"favMovies": "Inception",
"id": "c92e0e2a-c9f4-43ff-891b-a2561c6ce4d0"
}
Update user name by uuid /users/:id PUT A JSON object holding data from the user to be updated:
{
"name": "Updated name"
} 		A JSON object holding data from the user that has been updated:
{
"user": "2",
"name": "Updated name",
"favMovies": []
}
Update users favMovies by title and user id /users/:id/:title POST None A message returning if a movie has been successfully added to the list.
Delete a user's favMovie /users/:id/:title DELETE None A message returning if a movie has been successfully removed from the list.
Delete a user /users/:id/ DELETE None A message returning if the user was successfully deleted
